# 🏦 Kyung Hee University - Data Analysis Capstone Design

이 저장소는 경희대학교 데이터 분석 캡스톤 디자인 프로젝트를 위한 저장소입니다.  
본 프로젝트는 **금융권의 추천 시스템**을 주제로 하며, 특히 **신파일러(Thin Filer)** 모색을 통한 고객 유치 전략을 탐구합니다.

---

## 🎯 프로젝트 개요
*   **주제**: 금융권 추천 시스템을 활용한 새로운 고객층(신파일러) 모색 및 사용자 중심 금융 서비스 구축
*   **핵심 가치**: 리스크 최소화, 신뢰 기반 서비스, 사용자 중심의 경험 제공

## 🔍 문제 정의 및 배경
*   **금융권의 과제**: 기존 공급자 중심의 금융 시스템에서 사용자 중심으로의 변화가 필요함.
*   **전환 전략**: 기존 금융에 대한 신뢰를 바탕으로, 타 고객을 뺏어오기보다는 금융 이력이 부족한 **신파일러(소상공인, 저신용자 등)**를 타겟으로 하여 새로운 고객층을 확보함.
*   **목표**: 접근성 높은 은행(예: 토스뱅크 모델)과 같이 다양한 사용자층이 부담 없이 이용할 수 있는 추천 시스템 구현.

## 🛠️ 분석 프로세스
1.  **과제 정의**: 금융권 리스크 관리 및 신뢰 기반의 추천 시스템 정의
2.  **가설 수립**: 신파일러 대상의 맞춤형 금융 상품 추천의 효과성 검증
3.  **데이터 설계/수집/가공**: (예정)
4.  **알고리즘 선정**: 추천 시스템 알고리즘 도입
5.  **학습 및 파라미터 튜닝**
6.  **시스템 구현 및 평가**
